Paris21 Prototype
================

[![Build Status](https://travis-ci.org/devgateway/paris21.svg?branch=master)](https://travis-ci.org/devgateway/paris21)

## Requirements

* IOJS >= 3 or NodeJS >= 4

## Demo
http://devgateway.github.io/paris21

## Installation

```bash
$ npm install
```

## Build

```bash
$ npm run build
```

## Development

### Run the dev server

```bash
$ npm start
```

Will serve the app at [`localhost:3000`](http://localhost:3000).

### Lint and test

```bash
$ npm test
```

#### Lint only

```bash
$ npm run lint
```

#### Test only

```bash
$ npm run test-only
```
#### Publish gh-pages
```bash
$ npm run deploy
```
